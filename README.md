# Jaya Aishwarya — Portfolio

A modern, responsive portfolio built with Next.js, TypeScript, and Tailwind CSS to showcase frontend engineering work, selected projects, and technical experience.

## Live Demo

[View Portfolio](https://portfolio-lemon-tau-exng2nc4f4.vercel.app/)

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lottie React
- Lucide Icons

## Features

- Responsive design
- Light / dark mode
- Smooth animations
- Reusable component architecture
- Project showcase
- Resume download
- Contact section
- SEO metadata + Open Graph support

## Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Design Notes

The portfolio follows a clean component-driven architecture with a premium Dusty Plum visual theme, subtle motion, and a focus on readability, performance, and modern frontend UX patterns.
